# 🚀 MERN Stack Project – Product/User Management App

## 📌 Overview

This is a simple **MERN Stack application** that allows users to:

* Add data (Product/User)
* Store it in MongoDB
* Fetch and display it using React

It demonstrates the basics of full-stack development using MongoDB, Express, React, and Node.js.

---

## 🛠️ Tech Stack

* **Frontend:** React.js
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **API Testing:** Axios

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/Anshhuc/mern-task1.git
cd mern-task1
```

---

### 2️⃣ Install backend dependencies

```
npm install
```

---

### 3️⃣ Start backend server

```
node server.js
```

---

### 4️⃣ Setup frontend

```
cd frontend
npm install
npm start
```
